# 🌸 Iris Flower Classification

A Machine Learning web application built using Python and Streamlit that predicts Iris flower species based on flower measurements.

## 🚀 Features

- Beautiful and interactive UI
- Real-time flower prediction
- Machine Learning based classification
- Data visualizations using Plotly
- User-friendly Streamlit web app

## 📊 Dataset

The project uses the famous Iris dataset containing:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

Target Classes:
- Iris-setosa
- Iris-versicolor
- Iris-virginica

## 🤖 Machine Learning Algorithm

- Logistic Regression

## 🛠 Technologies Used

- Python
- Streamlit
- Pandas
- Scikit-learn
- Plotly
